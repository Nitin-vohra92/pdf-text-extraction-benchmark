PDFExtract
================================================================================

Executed Steps:
--------------------------------------------------------------------------------
* Created directory:
	mkdir PDFExtract && cd PDFExtract
* Installed TEI P5 model:
	git clone http://github.com/elacin/TEI-P5-Java-model.git
	cd TEI-P5-Java-model/
	git checkout 29d668e
	mvn install
	cd ..
* Installed patched PDFBox (revision 1157684):
	svn checkout http://svn.apache.org/repos/asf/pdfbox/trunk/@1157684 pdfbox
	cd pdfbox
	patch -p0 < ../PDFExtract/parent/patch/pdfbox_poms.patch
	patch -p0 < ../PDFExtract/parent/patch/pdfbox-font-bounding-boxes.patch 
	patch -p0 < ../PDFExtract/parent/patch/pdfbox-drawer-visibility.patch
	mvn install
	cd ..
* Installed PDFExtract:
	git clone https://github.com/elacin/PDFExtract
	cd PDFExtract/model
	mvn install
	cd PDFExtract/datasource
	mvn install
	cd PDFExtract/parent
    !! see 'pdfextract-cli/pom.xml' for changes made to get a single jar with 
    all dependencies after install !! 
	mvn -DskipTests=true install
* Usage:
	cd PDFExtract/pdfextract-cli
    java -jar target pdfextract-cli-M3-SNAPSHOT-jar-with-dependencies.jar [...]
