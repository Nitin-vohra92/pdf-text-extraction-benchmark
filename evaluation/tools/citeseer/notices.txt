CiteSeerExtractor
================================================================================

Installation
--------------------------------------------------------------------------------

* Get the code:
    git clone https://github.com/SeerLabs/CiteSeerExtractor.git
* Install python-module web.py (! Works only with python2 !)
    (sudo apt-get install python-pip) -> To get pip for python2
    sudo pip2 install web.py
* Install python-module xmltodict:
    sudo pip2 install xmltodict
* Install python-module python-magic:
    sudo pip2 install python-magic
* Install perl-module String::Approx
    cpan String::Approx 
(Make sure you have gcc/g++ version of at least 5.2.1 installed:
    sudo apt-get install gcc-5 g++-5
    sudo update-alternatives 
    sudo update-alternatives --remove-all gcc
    sudo update-alternatives --remove-all g++
    sudo update-alternatives --install /usr/bin/gcc gcc /usr/bin/gcc-5 20
    sudo update-alternatives --install /usr/bin/g++ g++ /usr/bin/g++-5 20
    sudo update-alternatives --config gcc
    sudo update-alternatives --config g++
    cd /usr/bin
    sudo rm x86_64-linux-gnu-gcc
    sudo ln -s gcc-5 x86_64-linux-gnu-gcc
)   
* Install django (wtf!)
    sudo pip2 install django
* On 64-bit systems you'll need support for 32-bit applications:
    sudo apt-get install ia32-libs-multiarch

Modifications
--------------------------------------------------------------------------------
Needed modify a bit of code:

* Implemented src/command_line.py that allows to use the extraction of CiteSeerX
via commandline. CiteSeerX only provides an HTTP API out of the box.

* Needed to change line

    ROOT_FOLDER="../" # there must be a trailing /

in service.py, utilities.py and extraction.py to 

    FOLDER      = os.path.dirname(os.path.realpath(__file__))
    ROOT_FOLDER = os.path.dirname(FOLDER) + "/"

because the original variant doesn't allow to call the script from outside the
directory.    

Usage
--------------------------------------------------------------------------------

python src/command_line.py <PDF> <OUT>

!! Please note the command "python" and *not* python3. !!
