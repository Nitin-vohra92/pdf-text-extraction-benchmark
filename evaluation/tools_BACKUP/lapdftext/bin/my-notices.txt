lapdftextProject
================================================================================

Executed Steps:
--------------------------------------------------------------------------------

* Cloned lapdftextProject:
	git clone https://github.com/BMKEG/lapdftextProject
	git submodule update --init --recursive
* Adapted bmkeg-parent/pom.xml:
        - Moved repository "jsi.sourceforge.net" to the end of "repository"
          section (because it responds with pom.xml files containing 
          "Moved permanently" error messages although the repository doesn't 
          provide the package).
	- Removed (commented out) the plugin repository "oss.sonatype.org" 
	  (because it yielded to corrupt pom.xml files). 
* Adapted lapdftext/pom.xml:
	- Removed (commented out) plugin section about "maven-bundle-plugin"
	- Removed (commented out) plugin section about "maven-assembly-plugin"
        - Added own plugin section about "maven-assembly-plugin":
	  
          <plugin>
            <groupId>org.apache.maven.plugins</groupId>
            <artifactId>maven-assembly-plugin</artifactId>
            <version>2.4.1</version>
            <configuration>
              <!-- get all project dependencies -->
              <descriptorRefs>
                <descriptorRef>jar-with-dependencies</descriptorRef>
              </descriptorRefs>
            </configuration>
            <executions>
              <execution>
                <id>make-assembly</id>
                <!-- bind to the packaging phase -->
                <phase>package</phase>
                <goals>
                  <goal>single</goal>
                </goals>
              </execution>
            </executions>
          </plugin>

	- Added dependency:

	  <dependency>
            <groupId>net.sf.trove4j</groupId>
	    <artifactId>trove4j</artifactId>
	    <version>2.0.2</version>
	  </dependency>

* Installed lapdftextProject:
	mvn install

* Command to run:
	cd lapdftext
	java -classpath target/lapdftext-1.7.5-SNAPSHOT-jar-with-dependencies.jar edu.isi.bmkeg.lapdf.bin.BlockStatistics
