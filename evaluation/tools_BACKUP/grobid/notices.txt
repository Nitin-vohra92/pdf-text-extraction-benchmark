My Notices about Grobid
================================================================================

Installation
--------------------------------------------------------------------------------

* wget https://github.com/kermitt2/grobid/archive/grobid-parent-0.4.0.zip
* unzip grobid-grobid-parent-0.4.0.zip

* Modified code in grobid-core/src/main/java/org/grobid/core/engines/ProcessEngine.java, lines 143-153:

    File[] files = null;   
    if (pdfDirectory.isDirectory()) {
        files = pdfDirectory.listFiles();
    } else if (pdfDirectory.isFile()) {
        files = new File[1];
        files[0] = pdfDirectory;
    }
    
  and lines 194-197:

    if (outputPath != null && outputPath.endsWith(".txt")) {
        Utilities.writeInFile(outputPath, result.toString());
	    continue;
	}

  to be able to process single files.
    
* mvn clean install


Usage:
--------------------------------------------------------------------------------

java -Xmx1024m -jar ./bin/grobid/grobid-core/target/grobid-core-0.4.0.one-jar.jar 
    -gH ./bin/grobid/grobid-home/ 
    -gP ./bin/grobid/grobid-home/config/grobid.properties 
    -ignoreAssets 
    -dIn <PDF> -dOut <OUT>
    -exe processFullText
